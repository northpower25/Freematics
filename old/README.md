# old/

This directory contains older firmware versions that are no longer actively maintained but are kept here for reference.

| Directory | Hardware | Notes |
|---|---|---|
| `firmware_v2/` | ATmega328p (Freematics ONE) | Legacy AVR firmware |
| `firmware_v3/` | ATmega328p (Freematics ONE) | Legacy AVR firmware |
| `firmware_v4/` | ESP32 (Freematics ONE+) | Older ESP32 firmware; superseded by `firmware_v5/` |

The current firmware for the Home Assistant integration is in `../firmware_v5/telelogger/`.
